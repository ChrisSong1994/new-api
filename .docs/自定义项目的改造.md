# 自定义项目的改造

本文档记录了将项目改造为商业化版本所做的修改。

## 改造日期

2026-05-04

## 改造内容

### 1. README 文档简化

**修改文件：**
- `README.md` - 简化为英文版，移除开源相关内容
- `README.zh_CN.md` - 简化为中文版，移除开源相关内容

**移除的内容：**
- 项目 Logo 和徽章（license、release、docker、GoReportCard 等）
- Trendshift、HelloGitHub、Product Hunt 等开源项目展示徽章
- 合作伙伴展示区域（Cherry Studio、Aion UI、北京大学、UCloud、阿里云、IO.NET）
- 特别鸣谢区域（JetBrains 开源许可证）
- Star History 图表
- 贡献指南
- 上游项目和配套工具链接（One API、Midjourney-Proxy、neko-api-key-tool 等）
- 官方文档链接（docs.newapi.pro）
- 社区交流、问题反馈等开源社区相关链接
- AGPLv3 许可证说明和商业许可联系方式

**保留的内容：**
- 项目简介
- 快速开始指南（Docker 部署）
- 主要特性说明
- 部署要求和环境变量配置
- 商业许可证声明

### 2. 删除多余的 README 语言版本

**删除的文件：**
- `README.zh_TW.md` - 繁体中文版
- `README.en.md` - 英文版（重复）
- `README.fr.md` - 法语版
- `README.ja.md` - 日语版

**保留的文件：**
- `README.md` - 英文版（主文档）
- `README.zh_CN.md` - 简体中文版

### 3. i18n 配置简化 - web/default

**修改文件：**
- `web/default/src/i18n/config.ts`

**修改内容：**
- 移除 fr、ru、ja、vi 的语言导入
- `resources` 只保留 `{ en, zh }`
- `supportedLngs` 只保留 `['en', 'zh']`

### 4. 删除多余的 i18n locale 文件

**删除的文件（web/default）：**
- `locales/fr.json` - 法语
- `locales/ru.json` - 俄语
- `locales/ja.json` - 日语
- `locales/vi.json` - 越南语
- `locales/_reports/` - i18n 同步报告目录

**保留的文件：**
- `web/default/src/i18n/locales/en.json`
- `web/default/src/i18n/locales/zh.json`

### 5. 移除 Classic 前端

**删除的目录：**
- `web/classic/` - 整个 classic 前端目录

**修改的文件：**
- `main.go` - 移除 classic 前端的 embed 变量和注入逻辑
- `router/web-router.go` - 简化为只使用 default 前端
- `common/embed-file-system.go` - 移除主题切换相关代码
- `common/constants.go` - 默认主题改为 "default"
- `setting/system_setting/theme.go` - 默认前端改为 "default"
- `controller/option.go` - 验证只接受 "default" 作为有效主题值
- `makefile` - 移除 classic 前端相关的构建目标
- 前端 `system-info-section.tsx` - 前端主题选择器改为只显示 "default" 选项

## 改造后的语言支持

项目现在只支持两种语言：
- **English (en)** - 英文
- **Chinese (zh-CN/zh)** - 简体中文

## 后续建议

如需进一步改造，可以考虑：
1. 修改项目名称和品牌标识
2. 更换 Logo 图片
3. 修改版权声明和许可证文件
4. 移除或替换原项目的文档链接
5. 检查代码中的原项目引用（如 API 文档链接、GitHub 仓库链接等）
